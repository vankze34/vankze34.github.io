Programmers Solution - 다리를 지나는 트럭

Programmers 문제 링크 바로가기

문제 설명

[##_Image kage@cSNPRZ/btq6mHaCezx/3lFaOepZuFqYF1usHgLKGK/img.png alignCenter width=”100%” data-origin-width=”1255” data-origin-height=”962” data-ke-mobilestyle=”widthOrigin”     _##]

접근방식

  • 트럭은 동시에 탑승 할 수 없다 (직렬)
  • 다리의 길이의 단위는 1초이므로 모든 트럭은 다리의 길이 만큼의 초가 경과한 경우 다리를 지나갈 수 있다
  • [한번에 올라갈 수 있는 트럭의 무게의 합 <= 다리가 견딜 수 있는 하중] 이어야 한다
  • 더이상 다리에 트럭이 없어진 시간이 return된다 (다리 위의 트럭을 나타내는 배열이 비어있어야 한다)
  • 가장 처음 들어간 트럭이 가장 먼저 나오므로 Queue의 개념으로 접근한다

예) 다리의 길이(초)가 2, 다리가 견딜 수 있는 하중: 10, 트럭의 무게: [7, 4, 5, 6] 의 경우
※ [0]은 트럭이 타지 않은 빈 공간으로 한다 / 트럭은 오른쪽에서 왼쪽으로 이동

경과시간 다리를 건넌 트럭 다리를 건너는 트럭 대기 트럭
초기 [] [0, 0] [7, 4, 5, 6]
1 [] [0, 7] [4, 5, 6]
2 [] [7, 0] [4, 5, 6]
3 [7] [0, 4] [5, 6]
4 [7] [4, 5] [6]
5 [7, 4] [5, 0] [6]
6 [7, 4, 5] [0, 6] []

알고리즘

def solution(bridge_length, weight, truck_weights):
    answer = 0
    q = [0] * bridge_length                            # 다리의 길이만큼 리스트를 생성
    while q:                                        # q가 비어있지 않은 동안 반복
        answer += 1                                    # 1초씩 증가
        q.pop(0)                                    # q의 0번 index를 제거 (다리를 건넘)
        if truck_weights:                            # 대기중인 트럭이 있다면
            if sum(q) + truck_weights[0] <= weight: # [다리위의 트럭의 무게 + 첫번째 대기중인 트럭의 무게 <= 다리의 하중] 인 경우
                q.append(truck_weights.pop(0))        # 첫번째 대기중인 트럭을 대기열에서 빼서 q에 추가
            else:                                    # [다리위의 트럭의 무게 + 첫번째 대기중인 트럭의 무게 <= 다리의 하중] 이 아닌 경우
                q.append(0)                            # q에 0을 추가 (트럭이 타지 않음)

    return answer
