Programmers Solution - 소수찾기

Programmers 문제 링크 바로가기

문제 설명

접근방식

  • 각 수포자가 찍는 방식들의 규칙이 존재한다
  • 최다 득점 점수와 최다 득점을 한 사람을 출력을 알아야 return 할 수 있다

알고리즘

from itertools import permutations
def solution(numbers):
    n = len(numbers)
    p_numbers = []
    cases = []
    answer = 0
    cases_store = set()
    
    for i in range(1, 2**n):
        number_flag = str(bin(i)[2:]).zfill(n)
        cases.append(number_flag)
    
    for number in permutations(numbers, n):
        tmp = ""
        for i in number:
            tmp+=i
        p_numbers.append(tmp)
    
    for num in p_numbers:
        for l in cases:
            tmp=""
            for i, s1 in enumerate(l):
                if s1!="0":
                    tmp+=str(int(s1)*int(num[i]))
            cases_store.add(int(tmp))
        cases_num = cases_store.copy()
    for i, s in enumerate(cases_store):
        if s<2:
            cases_num.remove(s)
        elif s>2:
            for i in range(2, s):
                if s % i == 0:
                    cases_num.remove(s)
                    break
    return len(cases_num)
