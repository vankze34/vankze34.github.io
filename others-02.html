CentOS의 Jupyter Notebook 외부에서 접속하기

선행: CentOS에 Jupyter Notebook 설치

프로파일 생성

$ jupyter notebook --generate-config

위의 명령어를 통해 ‘/home/(username)/.jupyter/jupyter_notebook_config.py’파일이 생성된다

내부ip 확인

$ ifconfig

        inet [확인해야 할 ip주소]  netmask 255.255.240.0  broadcast 172.31.47.255
        inet6 fe80::85e:e5ff:fe32:421a  prefixlen 64  scopeid 0x20<link>
        ether 0a:5e:e5:32:42:1a  txqueuelen 1000  (Ethernet)
        RX packets 38282  bytes 19365933 (18.4 MiB)
        RX errors 0  dropped 0  overruns 0  frame 0

[확인해야 할 ip주소] 를 기록해둔다

프로파일 수정

$ vi /home/(username)/.jupyter/jupyter_notebook_config.py

ip address 수정 후 주석 제거

# /home/(username)/.jupyter/jupyter_notebook_config.py

(중략)

c.NotebookApp.ip = '[위에서 기록해놓은 ip주소 입력]'

jupyter notebook 실행 후 web-browser 실행 안되게 변경 후 주석 제거

# /home/(username)/.jupyter/jupyter_notebook_config.py

(중략)

c.NotebokkApp.open_browser = False

(후략)

포트번호 설정 후 주석 제거

# /home/(username)/.jupyter/jupyter_notebook_config.py

(중략)

c.NotebookApp.port = XXXX

(후략)

비밀번호 설정

$ ipython

from IPython.lib import passwd
passwd()

비밀번호를 2회 입력 후 출력된 해시(‘sha1:…’) 복사

# /home/(username)/.jupyter/jupyter_notebook_config.py

(중략)

# c.NotebookApp.password = '위에서 복사한 hash 입력'

(후략)

서버에서 jupyter notebook 실행

$ jupyter notebook

이후 클라이언트에서 http://서버ip주소:8000 로 접속
